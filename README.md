# üíå Valentine's Day Digital Card

A simplified, interactive, and responsive web-based Valentine's Day card. This project replaces the traditional paper card with a digital experience featuring animations, playful interactivity, and a romantic aesthetic.

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## ‚ú® Features

* **Interactive "Yes/No" Buttons**: The core interaction where the user is asked to be your Valentine.
* **Playful Logic**:
    * If the user clicks **"Yes"**: A celebration screen appears with confetti/animations.
    * If the user tries to click **"No"**: The button moves away, shrinks, or changes text to playfully convince them otherwise.
* **Responsive Design**: Works seamlessly on mobile phones, tablets, and desktops.
* **Custom Animations**: Smooth CSS transitions for heart beats and button hovers.
* **Audio Support**: (Optional) Background music can be toggled on/off.
